Autonomous Disinfection Robot

Western Engineering Competition: 1st Place Senior Design Category

Designed an autonomous search and rescue robot CAD model and designed a prototype pathfinding robot. The robot uses ultrasonic sensors to gage distance from obsticals. An arduino handles the code that allows the robot to correct its path in real time.

IR Beacon detection robot

Mechatronics Design Course

This robot was tasked with locating a beacon in an enclosed area and to use its front bumper to disable it. The track has two seperate beacons each giving off a different IR signal, the robot uses an IR sensor and blinder combo to accurately find source. An arduino is used to decode the signal so the robot aproaches the correct beacon.
