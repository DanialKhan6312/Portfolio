Creating Art with a Cycle GAN

Western AI Specialized Project Team

GitHub Github

The goal of this work is to demonstrate a machine learning modelâ€™s ability to recreate creative tasks, using the Cycle Generative Adversarial Network (GAN) architecture to create artwork out of photographs. Approaching image translation with a Cycle GAN allows for the use of unpaired data while training, meaning we can generate images without having a correlated dataset to compare to. A Cycle GAN involves four neural networks making up the generator and discriminator components for two GANs, one to translate the image and one to revert to the original to assure cycle consistency. This is to assure that the output image does not disregard the input and translates the image rather than generate a random one. The results above are of two pictures translated into artwork.

Mercury: SMS based medical info app

Carelton University Hackathon: 1st Place Inclusivity Category

GitHub Github

Mercury works to service the population of the world that has access to mobile services but not a reliable internet connection. The Goal of Mercury is to use SMS to deliver a convenient medical search app to anyone in the world even without the luxury of an internet connection. Using APIs such as Twilio, Endlessmedicine, and BeautifulSoup, Mercury uses a Python Flask Backend server to process the incoming SMS data and scan the web using a BeautifulSoup web scrapper. This info is then summarized using a Natural Language Processing import and returns concise information on possible conditions faced by the user.
